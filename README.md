# **一个 CS:GO 27年电竞传奇游戏宣传网站**

---

## **一、创意名称 + 创意介绍**

<img width="1920" height="1167" alt="image" src="https://github.com/user-attachments/assets/04d79289-0386-4bc7-86c2-c60bb0c5871c" />


**创意名称：**

**CS:GO 27年电竞传奇游戏宣传网站**

**想解决什么问题：**

<img width="1920" height="1167" alt="image" src="https://github.com/user-attachments/assets/7b3d2325-6429-404b-a897-8f941223c6f2" />


  大多数玩家对 Counter-Strike 这款游戏的情感记忆停留在碎片化的论坛帖子和视频片段里，缺少一个视觉精良、叙事完整的"荣耀档案馆"来系统呈现这款跨越27年的电竞传奇。

**为什么会想到做这个：**
<img width="1920" height="1167" alt="image" src="https://github.com/user-attachments/assets/e6c7e28a-a56c-4c77-b861-0eecfabef254" />

  2026年是 CS 系列诞生27周年，官方贺词"感谢每一位投身对局的玩家"触动了无数 CSer。作为一名长期玩家，我想用一个有质感的网页，把这27年的历史节点、经典特色、电竞荣耀和数据成就一次性呈现给每一位路人和老玩家，让更多人感受到这款游戏的文化厚度。

**大概是什么产品：**

  一个基于纯 HTML / CSS / JavaScript 制作的单页游戏宣传网站，无需后端，开箱即用，可直接部署或分享。

---

## **二、目标用户及痛点**

**面向哪些用户：**

* CS 系列游戏的现役玩家与退坑老玩家

* 对电竞文化感兴趣的泛游戏人群

* 希望快速了解 CS:GO 核心特色的新玩家

**在什么场景下使用：**

* 玩家向朋友安利 CS 游戏时，直接分享这个网页代替口头介绍

* 电竞社团/网吧/线下赛事活动现场展示

* 个人项目展示、简历附件（网页技术作品集）

**当前痛点：**

  没有这个产品之前，玩家想系统介绍 CS 的历史与魅力，只能拼凑百度百科、B站视频、Steam页面，信息分散、缺乏沉浸感，且没有统一的视觉表达能让外行人一眼就感受到 CS 的硬核气质。

---

## **三、价值与意义**

**文化价值：**

<img width="1920" height="1167" alt="image" src="https://github.com/user-attachments/assets/99342048-f7f7-4a4e-aeca-e6b398dbb384" />


  Counter-Strike 是全球最具影响力的电竞 IP 之一，全球超 19,000 场锦标赛、逾 2.51 亿美元赛事奖金。这个网站以时间轴、数据卡、选手名人堂等形式，将27年电竞历史可视化，是一份有温度的电竞文化档案，有助于推动游戏文化的记录与传播。

**效率提升：**

  网站采用单文件 HTML 架构（无框架依赖），任何人只需打开一个 `.html` 文件就能看到完整内容，极大降低了信息获取门槛。固定导航、平滑滚动、响应式设计使桌面和移动端都能获得流畅的浏览体验。

## **四、产品亮点（附加说明）**

| **模块** | **内容** |
|:---|:---|
| 英雄区 | Canvas 粒子动画 + CS.png 官方素材背景 + 战术网格 |
| 游戏简介 | CT vs T 阵营对比卡片 + 四大经典模式展示 |
| 核心特色 | 6张悬停动效卡片（战术/经济/地图/皮肤/匹配/Major） |
| 27年时间线 | 垂直双栏布局，10个历史节点（1999→2026） |
| 电竞传奇 | 5位传奇选手 + 中国荣耀 wNv 2005年夺冠专题 |
| 数据统计 | 滚动触发计数动画（27年/5版本/19K+赛事/$251M奖金） |

**技术特性：**

* 纯原生 HTML + CSS + JavaScript，零框架依赖

* 滚动揭示动画（淡入 / 左右滑入）

* 自定义鼠标准星跟随效果

* 顶部阅读进度条

* 完全响应式，适配桌面 / 平板 / 手机
